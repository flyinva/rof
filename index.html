Flux RSS Ouest-France

Le site Ouest-France ne propose pas de lien RSS pour les informations de chaque commune. Ce site propose de le générer à partir des pages Ouest-France. Pour cela, tapez le nom de votre commune, un lien vers le RSS apparaîtra.

Ce site n'a aucun lien avec Ouest-France. Le code source de l'application est disponible sous licence WTFPL. Auteur : Sylvain
